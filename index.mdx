## Quick Start

Get started with the Tourshub API in just a few steps:

<Steps>
  <Step title="Get API Credentials">
    Contact [support@tourshub.org](mailto:support@tourshub.org) to obtain your API credentials and access tokens.
  </Step>
  <Step title="Choose Your Integration">
    Select the appropriate API based on your use case:

    - **V1 API** for standard booking applications
    - **Experience Bank Partner API** for tour operator integration
    - **Internal APIs** for system-to-system communication
  </Step>
  <Step title="Make Your First Request">
    Start with a simple tour search request to test your integration.
  </Step>
  <Step title="Implement Booking Flow">
    Build your booking workflow using our shopping cart and reservation APIs.
  </Step>
</Steps>

## Authentication Methods

Tourshub API supports multiple authentication methods depending on your use case:

| Method              | Use Case                     | Documentation                                             |
| ------------------- | ---------------------------- | --------------------------------------------------------- |
| **JWT Tokens**      | Standard API access          | [V1 API](/api-reference/v1-api)                           |
| **Private Tokens**  | Sensitive booking operations | [Booking API](/api-reference/booking-api)                 |
| **HMAC Signatures** | Partner integrations         | [Experience Bank API](/api-reference/experience-bank-api) |
| **Basic Auth**      | Internal system access       | [Intranet API](/api-reference/intranet-api)               |

## API Endpoints Overview

### V1 API (`/api/`)

- Tour search and filtering
- Location and currency data
- Authentication token management

### Booking API (`/api/booking/`)

- Shopping cart management
- Reservation creation and management
- Payment processing
- Billing data handling

### Experience Bank Partner API (`experiencebank.domain/`)

- Activity listings for partners
- Availability checks
- Partner booking management
- Cancellation handling

### Internal APIs

- **Intranet API**: Data synchronization and order management
- **Tour CMS Webhooks**: Content management and booking notifications

<Note>
  All APIs support both sandbox and production environments. Contact our support team for access to the appropriate environment for your needs.
</Note>
